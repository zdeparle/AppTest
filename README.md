# MechanicalCalculatorApp

This is a simple SwiftUI iPhone application that lets you perform a few basic mechanical engineering calculations.

## Features

- Select an equation: Newton's second law (`F = m × a`), Torque (`τ = F × r`), or Pressure (`P = F / A`).
- Choose which variable to solve for within the selected equation.
- Enter the remaining variables and tap **Calculate** to compute the result.

## Building

Open the `MechCalc` directory in Xcode as a new project (SwiftUI App template) and copy the files from this repository. Build and run on the iOS simulator or device.
